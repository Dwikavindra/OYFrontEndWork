Variasi Fitur pada OY! Payment Link

Atur Jangka Waktu Aktif Link

Pilih tipe Payment Link yang Anda butuhkan: One-Time untuk penggunaan sekali atau Reusable untuk penggunaan berulang (wajib berulang) yang berlaku selamanya. Tipe Reusable dapat digunakan untuk berbagai transaksi tanpa periode kedaluwarsa, sedangkan tipe One-Time terdapat periode kedaluwarsa.

‍

Atur Jumlah Pembayaran

Pilih tipe pengaturan jumlah pembayaran yang Anda inginkan: Open Amount (jumlah pembayaran fleksibel sehingga pelanggan dapat mengisi jumlah pembayaran sendiri) atau Closed Amount (jumlah pembayaran diatur oleh Anda agar pelanggan dapat langsung melakukan pembayaran).

‍

Atur Tampilan Halaman Pembayaran Sesuai Branding
Bisnis

Untuk mempertahankan branding bisnis Anda, Anda dapat mengatur tampilan halaman pembayaran payment link melalui dashboard. Klik untuk info lanjut.

‍

Pengiriman Payment Link melalui Email dan/atau
WhatsApp

Anda akan mendapatkan notifikasi berisi payment link untuk pelanggan Anda sebagai pengingat untuk melakukan pembayaran. Fitur ini dapat meningkatkan kepercayaan pelanggan karena OY! Indonesia adalah penyedia layanan keuangan terbaik untuk bisnis Anda.

Berbagai Pilihan Tipe Integrasi OY! Payment Link

via Desktop atau Mobile Dashboard

Tanpa integrasi, Anda dapat menikmati produk OY! Payment Link untuk bisnis Anda. Pilih platform yang diinginkan, mulai dari akses melalui dashboard laptop/komputer, atau pun melalui handphone untuk kemudahan akses.

‍

via Integrasi

OY! Payment Link tersedia dengan solusi integrasi melalui API. Dengan integrasi API yang cepat dan mudah, Anda dapat membuat payment link melalui platform Anda secara otomatis saat pelanggan ingin melakukan pembayaran.

‍

via Routing Solution

Dengan satu integrasi sederhana (sistem routing), Anda dapat mendapatkan dan menggunakan produk OY! Payment Link dan OY! Multitransfer secara bersamaan. Sistem routing juga memungkinkan untuk setiap kali Anda menerima uang, saldo akan secara otomatis dikirimkan hingga ke 10 rekening penerima.

Online Shop

Bagikan OY! Payment Link melalui media sosial dan aplikasi messenger (contoh: WhatsApp, Telegram, Facebook Messenger) untuk terima pembayaran pesanan dari pelanggan.

Offline Shop

Bagikan OY! Payment Link melalui media sosial dan aplikasi messenger untuk terima pembayaran pesanan online atau delivery.

Freelancer

Freelancer atau pekerja lepas dapat menggunakan OY! Payment link untuk terima pembayaran dari klien melalui email, aplikasi messenger, dan media sosial (contoh: Instagram, Facebook, TikTok, LinkedIn) untuk proses bertransaksi yang lebih profesional.

eCommerce

OY! Payment Link dapat terhubung dengan platform e-Commerce pada halaman pembayaran (checkout page) untuk tampilan halaman pembayaran yang lebih baik.

Payment Point Online Banking (PPOB)

Penyedia jasa PPOB dapat menggunakan OY! Payment Link untuk pembayaran top up voucher game, pulsa, tagihan, atau transfer uang ke suatu penerima (agen transfer) melalui aplikasi messenger atau platform PPOB milik Anda sendiri.

Grosir Online

Toko grosir online dapat menggunakan OY! Payment Link untuk pembayaran online. Terdapat opsi pembayaran kartu debit, kartu kredit, dan transfer bank yang dapat menjadi solusi untuk kebutuhan pembayaran dalam jumlah besar.

Situs Web (Website Enabler)

OY! Payment Link memudahkan pihak pengembang situs web seperti vendor pembuatan situs bisnis/personal, hingga platform undangan online untuk mengintegrasikan produk ini langsung pada situs yang sedang dikerjakan untuk klien Anda.

Software Developer (SaaS)

Platform jasa pengembangan software (SaaS) dapat menggunakan OY! Payment Link sebagai opsi pembayaran untuk kelancaran pelanggan saat bertransaksi.
