Berbagai Pilihan Tipe Integrasi OY! Multitransfer

via Integrasi (API Multitransfer)

Dengan integrasi API yang simpel dan cepat, OY! Multitransfer langsung terkoneksi dengan berbagai bank dan e-Wallet. Kirim dana langsung melalui sistem atau platform Anda dengan opsi URL Callback untuk proses integrasi dengan OY! yang lebih mudah.

via Dashboard (Bulk Multitransfer)

Tanpa integrasi, Anda dapat langsung melakukan transfer melalui dashboard. Cukup dengan menggunggah file .CSV atau isi informasi transaksi yang Anda lakukan di dashboard OY! Bisnis.

via Routing Solution

Satu integrasi sederhana untuk mendapatkan akses payment link dan produk disbursement. Sistem Routing memungkinkan Anda untuk mengirim dana ke rekening tujuan secara otomatis, hingga ke 10 penerima.

Tim Finance / Keuangan pada Setiap Jenis Industri

OY! Multitransfer akan memenuhi kebutuhan transaksi payroll atau pengiriman gaji karyawan dari perusahaan apa pun.

Perusahaan Finansial - Investasi

Dengan OY! Multitransfer, perusahaan investasi akan mengalami kemudahan dalam melakukan pembayaran dana user serta melimpahkan dana ke bank kustodian (contoh: Bibit).

Perusahaan Finansial - Kredit / P2P Lending

OY! Multitransfer akan membantu mendistribusikan dana kredit kepada para peminjam dari perusahaan kredit/P2P  lending (contoh: Pegadaian, Investree, Koinworks).

Payment Point Online Banking (PPOB)

Penyedia jasa PPOB dapat menggunakan OY! Multitransfer untuk mengirim dana ke tujuan yang diminta dan melakukan top-up saldo e-Wallet.

eCommerce / Marketplace

Platform eCommerce atau marketplace dapat mengirim dana hasil transaksi kepada merchant/mitra/agen secara efisien serta mengirim hasil penjualan ke distributor/vendor. Selain itu, pembagian hadiah, cashback, cash, dan saldo e-Wallet juga dapat dikirim melalui OY! Multitransfer.

FMCG / Retail / Logistik (Tokopedia, gudangada, ids, orderkuota)

Perusahaan FMGC / Retail / Logistik dapat menggunakan OY! Multitransfer untuk mengirim dana ke para distributor dengan mudah (contoh: Tokopedia, Gudangada, IDS, Orderkuota).

Toko Online (Online Shop)

Para pemilik bisnis online shop mampu memakai OY! Multitransfer untuk melakukan pembayaran ke supplier atau vendor serta membagikan hadiah berupa saldo (cash) kepada pelanggan.

Online Shop

Offline Shop

Bagikan OY! Payment Link melalui media sosial dan aplikasi messenger untuk terima pembayaran pesanan online atau delivery.

Freelancer

Freelancer atau pekerja lepas dapat menggunakan OY! Payment link untuk terima pembayaran dari klien melalui email, aplikasi messenger, dan media sosial (contoh: Instagram, Facebook, TikTok, LinkedIn) untuk proses bertransaksi yang lebih profesional.

eCommerce

OY! Payment Link dapat terhubung dengan platform e-Commerce pada halaman pembayaran (checkout page) untuk tampilan halaman pembayaran yang lebih baik.

Payment Point Online Banking (PPOB)

Penyedia jasa PPOB dapat menggunakan OY! Payment Link untuk pembayaran top up voucher game, pulsa, tagihan, atau transfer uang ke suatu penerima (agen transfer) melalui aplikasi messenger atau platform PPOB milik Anda sendiri.

Grosir Online

Toko grosir online dapat menggunakan OY! Payment Link untuk pembayaran online. Terdapat opsi pembayaran kartu debit, kartu kredit, dan transfer bank yang dapat menjadi solusi untuk kebutuhan pembayaran dalam jumlah besar.

Situs Web (Website Enabler)

OY! Payment Link memudahkan pihak pengembang situs web seperti vendor pembuatan situs bisnis/personal, hingga platform undangan online untuk mengintegrasikan produk ini langsung pada situs yang sedang dikerjakan untuk klien Anda.

Software Developer (SaaS)

Platform jasa pengembangan software (SaaS) dapat menggunakan OY! Payment Link sebagai opsi pembayaran untuk kelancaran pelanggan saat bertransaksi.
